
# 日期验证装饰器

| 装饰器                                              | 描述                                                                                                                                                                                           |
| ------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `@MinDate(date: Date \| (() => Date))`                 | 检查该值是否为指定日期之后的日期。                                                                                                                                        |
| `@MaxDate(date: Date \| (() => Date))`                 | 检查该值是否为指定日期之前的日期。                                                                                                                                       |
