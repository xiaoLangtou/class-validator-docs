
# 字符串类型验证装饰器

| 装饰器                                              | 描述                                                                                                                                                                                           |
| ------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `@IsBooleanString()`                                   | 检查字符串是否为布尔值（例如，为 “true” 或 “false” 或 “1”、“0”）。                                                                                                                              |
| `@IsDateString()`                                      | Alias for `@IsISO8601()`.                                                                                                                                                                             |
| `@IsNumberString(options?: IsNumericOptions)`          | 检查字符串是否为数字。                                                                                                                                                                       |
